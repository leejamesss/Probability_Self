# Probability_Self
概率与统计自学仓库

## 介绍

这是一个用于记录学习概率与统计数学知识的仓库。本仓库将包含概率论、统计学、随机过程等相关领域的基础知识、经典问题、常用方法以及实际应用等内容。旨在帮助初学者建立完整的知识体系，同时也欢迎高手指正。

- [统计学](./statistics)
